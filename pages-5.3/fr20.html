快照取证机制
OpenClaw Injection Snapshot — 捕获实例的完整渲染配置状态，包含 SelectedResourceIDs、ResolvedResources、RenderedManifest、RenderedEnv。
Instance Config Revision — 每次配置变更生成新修订版本，含 SHA256 Checksum 验证，ContentJSON 标记 json:"-" 防止 API 暴露。
Backup — 支持手动/定时备份，含 BackupSchedule CronExpression 和 RetentionDays。
