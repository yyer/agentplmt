数据加密存储
加密存储架构
系统采用多层加密策略：
1. K8s Secret — API Key、凭据等敏感配置存储在 K8s Secret 中，由集群 etcd 加密。
2. 凭据代理 — AI Gateway 通过 SecretRefService 在代理时解析 K8s Secret，应用层不持久化明文。
3. .env 加密 — SecureClaw 凭据加固模块将 .env 加密为 .env.enc（AES-256-GCM）。
4. 字段级保护 — 数据库模型中 json:"-" 标签阻止敏感字段 API 暴露。
