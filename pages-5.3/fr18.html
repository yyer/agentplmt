会话隔离
会话隔离架构
多层隔离确保 Agent 会话数据不互相泄露：
1. K8s Namespace 隔离 — 每个 Agent 运行在独立用户命名空间 clawreef-user-{userID}，Secret、Pod 等资源隔离
2. 会话令牌隔离 — agt_sess_ 会话令牌绑定实例，24h 滚动续期
3. Exfiltration Guard — 检测跨会话数据外泄链路（source → transform → sink）
4. KubeArmor 网络策略 — nftables 默认拒绝，仅白名单放行
