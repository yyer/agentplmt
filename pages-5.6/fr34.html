Risk Assessment
风险评估标志 (assessScriptArtifactRiskFlags)
6 项标志
script-high-risk-command Critical
解码文本匹配 HIGH_RISK_COMMAND_PATTERNS，检测高危系统命令
rm -rf
mkfs
dd if=
chmod 777
script-outbound-sink High
解码文本匹配外联模式，检测数据外传通道
curl
wget
nc
scp
script-secret-exfiltration Critical
秘密引用 + 外联组合，检测敏感数据外泄行为
$API_KEY
$SECRET
+ curl
script-encoded-high-risk-command Critical
编码后解码为高危命令，检测通过 Base64/Hex 编码绕过的高危操作
base64 -d
xxd -r
\x
script-encoded-secret-exfiltration Critical
编码后解码为秘密外泄，检测编码掩盖的敏感数据泄露
echo $KEY | base64
openssl enc
script-encoded-outbound-sink High
编码后解码为外联，检测编码掩盖的网络外联行为
eval $(base64)
python -c import

Pre-Execution Check
执行前校验流程
before_tool_call Strategy #8

在 before_tool_call 钩子中，script_provenance_guard (strategy order 8) 拦截 exec / bash 工具调用时，检查目标脚本路径是否在 Script Artifact 记录中：

exec / bash 工具调用
➜
查询 Script Artifact 记录
✗ 脚本在 Artifact 记录中
→ 检查风险标志 (riskFlags)
→ 有阻断标志 ➜ 阻止执行
→ 无阻断标志 ➜ 记录告警 + 审批
✓ 脚本不在 Artifact 记录中
→ 非AI生成脚本
→ 正常执行（交由其他策略检查）
