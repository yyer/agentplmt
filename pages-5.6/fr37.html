Dangerous Execution Mechanisms
动态执行机制 (需要禁用或限制)
Enforce 8 机制
机制语言危险等级状态说明
eval() Python / JS / Ruby CRITICAL 禁用 动态代码执行
exec() Python CRITICAL 禁用 动态代码执行
execve() C / Shell HIGH 限制 进程替换
pickle.load() Python CRITICAL 禁用 不安全反序列化
yaml.load() Python HIGH 限制 不安全反序列化 (需 yaml.safe_load)
subprocess shell=True Python HIGH 限制 Shell 注入风险
new Function() JavaScript HIGH 限制 动态函数构造
-enc / -encodedcommand PowerShell CRITICAL 禁用 编码命令执行
CRITICAL 等级机制已被完全禁用，HIGH 等级机制已被限制 (仅允许特定安全用法)
