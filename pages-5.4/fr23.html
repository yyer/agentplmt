模型完整性校验机制

ClawManager — SkillBlob 使用 content_hash（MD5 目录确定性序列化）和 archive_hash（SHA-256 标准化 ZIP）进行去重和完整性验证
Instance Config Revision — 每次配置变更生成 SHA256 Checksum，ContentJSON 标记 json:"-" 防止 API 暴露
KubeArmor — FIM 策略监控 /bin/、/usr/bin/、/sbin/、/boot/ 等目录的只读完整性
